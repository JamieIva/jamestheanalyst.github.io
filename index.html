News website

I designed and developed a fully responsive news website that serves as a comprehensive platform for delivering real-time news updates to users. The project is built using C# using Visual Studio, providing a solid and scalable back-end architecture. Leveraging The Guardian API, the website dynamically fetches and displays the latest news articles, covering a wide range of categories such as world news, sports, technology, and entertainment.

The integration with The Guardian's API involved handling JSON data, managing API requests, and ensuring efficient data parsing to present content seamlessly to users. I implemented features like category filtering, search functionality, and pagination to enhance user experience and navigation.

The website also includes a clean, user-friendly interface that adapts to various screen sizes, ensuring accessibility across devices. This project showcases my proficiency in C#, my ability to work with third-party APIs, and my skills in creating dynamic, data-driven web applications. Through this project, I demonstrated my commitment to delivering high-quality, functional web solutions that meet user needs.

Ticket Sales

I developed a responsive web application for ticket sales using C# in Visual Studio, with a MySQL database for data management. This project involved creating a user-friendly interface where users can browse events, view ticket availability, and make purchases. The web page is fully responsive, ensuring a smooth and consistent experience across various devices, including desktops, tablets, and mobile phones.

In this project, I utilized C# for backend development, managing tasks such as handling event data, ticket inventory, and processing transactions securely. The MySQL database was integrated to store and manage all relevant data efficiently. The front-end was designed to be intuitive and responsive, providing real-time updates on ticket availability and a seamless purchasing process. This project showcases my ability to develop a full-stack application using C#, integrate it with a MySQL database, and create a responsive design that enhances user interaction and experience.
