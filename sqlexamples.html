SQL statement examples

Have a look at how great I am with SQL. Be amazed!

All examples on this page use data from ourworldindata.org, namely data in covid deaths and vaccinations.

Skills used: Joins, CTE's, Temp Tables, Windows Functions, Aggregate Functions, Creating Views, Converting Data Types

Simple select

First let's look on some simple statements boobies i love:

SELECT * FROM covidbase.tblcoviddeaths WHERE location like '%Ghana%';
SELECT SUM(new_deaths) FROM covidbase.tblcoviddeaths WHERE location = 'Ghana';

Average life expectancy in Asia and Africa

Not as simple one now:

SELECT AVG(life_expectancy) as 'life expectancy', 
  continent, location, MAX(date)
FROM covidbase.tblcovidvaccinations
where continent = 'asia' or continent = 'africa'
group by continent, location
order by continent, location;

Total Cases vs Total Deaths

Shows likelihood of dying if you contract covid in your country

Select Location, date, total_cases,total_deaths, 
(total_deaths/total_cases)*100 as DeathPercentage
From covidbase.tblcoviddeaths
Where location like '%states%'
and continent is not null 
order by 1,2;
